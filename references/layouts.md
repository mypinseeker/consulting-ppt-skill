# McKinsey Page Layout Library

7 classic McKinsey page layouts and when to use them.

## 1. Title + Single Chart

```
+-------------------------------------+
| Action title (conclusion sentence)  |
+-----------+-------------------------+
|           |                         |
|  CHART    |     (60-70% space)      |
|           |                         |
+-----------+------------+------------+
| Source: ...            | INSIGHT    |
+------------------------+------------+
```

**Use when**: Single data story, trend, comparison
**Chart types**: Bar, line, area

## 2. Title + Left/Right Split

```
+-------------------------------------+
| Action title (conclusion sentence)  |
+-----------------+-------------------+
|                 |                   |
|  CHART or       |  TEXT or          |
|  BULLET LIST    |  SECOND CHART     |
|                 |                   |
+-----------------+-------------------+
```

**Use when**: Comparison, one-chart-one-text
**Common**: Chart + case study, metrics + explanation

## 3. Title + Chart + Table

```
+-------------------------------------+
| Action title (conclusion sentence)  |
+-----------------+-------------------+
|                 |                   |
|  STACKED BAR    |  DATA TABLE       |
|  CHART          |  (detail values)  |
|                 |                   |
+-----------------+-------------------+
| Source: ...                         |
+-------------------------------------+
```

**Use when**: Need both visual overview and precise numbers
**Example**: Cost dimension breakdown with exact values

## 4. Title + KPI Row + Chart

```
+-------------------------------------+
| Action title (conclusion sentence)  |
+--------+--------+--------+---------+
| KPI #1 | KPI #2 | KPI #3 | KPI #4  |
+--------+--------+--------+---------+
|                                     |
|           CHART (large)             |
|                                     |
+------------------------+------------+
| Source: ...            | INSIGHT    |
+------------------------+------------+
```

**Use when**: Executive summary, key metrics overview
**KPI boxes**: 3-4 max, each with value + label

## 5. Title + 2x2 Matrix

```
+-------------------------------------+
| Action title (conclusion sentence)  |
+------------------+------------------+
|                  |                  |
|   Quadrant 1     |   Quadrant 2    |
|                  |                  |
+------------------+------------------+
|                  |                  |
|   Quadrant 3     |   Quadrant 4    |
|                  |                  |
+------------------+------------------+
```

**Use when**: Strategic positioning, risk assessment
**Example**: Probability x Impact, Growth x Share

## 6. Title + Timeline

```
+-------------------------------------+
| Action title (conclusion sentence)  |
+-----+--------+------+--------------+
| Ph1 |  Ph2   | Ph3  |  -> Marker   |
+-----+--------+------+--------------+
|                                     |
| KPI BOX  |  COMPARISON BOX         |
|          |  (what the other path    |
|          |   achieves in same time) |
+---------+---------------------------+
```

**Use when**: Process phases, project timeline, delay analysis
**Phases**: Color-coded bars with duration labels

## 7. Title + Recommendation Cards

```
+-------------------------------------+
| Action title (conclusion sentence)  |
+---+----------+----------+----------+
| 1 | ACTION 1 | 2 ACTION | 3 ACTION |
+---+          |          |          |
|   Description|Description|Descript. |
|   box        |   box    |   box    |
+--------------+----------+----------+
|                                     |
|  SUPPORTING CHART or SUMMARY       |
+-------------------------------------+
```

**Use when**: Final recommendation, action plan
**Cards**: Numbered circles + title + description

## Layout Selection Decision Tree

```
Need key metrics overview?     -> KPI Row + Chart (#4)
Need trend or comparison?      -> Single Chart (#1)
Need chart + exact values?     -> Chart + Table (#3)
Need strategic positioning?    -> 2x2 Matrix (#5)
Need timeline/phases?          -> Timeline (#6)
Need final recommendation?     -> Recommendation Cards (#7)
Need two things side by side?  -> Left/Right Split (#2)
```
